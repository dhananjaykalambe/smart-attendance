Project Overview

{{ program_name }}

The system is designed to replace manual attendance collection with a cleaner digital experience. It gives faculty a fast way to launch attendance windows and helps students complete attendance with minimal friction.

Technology Stack

Core tools used

Flask SQLite HTML CSS JavaScript Python QRCode HTML5 QR Scanner

Features

What the system provides

  • QR-based attendance sessions with limited validity.
  • Duplicate attendance protection for every student-session pair.
  • Student directory management for cleaner record keeping.
  • Attendance records and subject-wise reporting pages.
  • Responsive interface suitable for desktop and mobile screens.
